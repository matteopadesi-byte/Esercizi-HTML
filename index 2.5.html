Glossario informatico

HTML
HyperText Markup Language, linguaggio per creare pagine web.
CSS
Cascading Style Sheets, linguaggio per lo stile delle pagine.
Browser
Programma per navigare in Internet; il programma serve a inoltrare la richiesta di un documento alla rete e ne consente la visualizzazione una volta arrivato.
Server
Un componente informatico con la funzione di elaborare e gestire i dati aspettando richieste dai client.
Database
Un insieme di informazioni (o dati) strutturate in genere archiviate elettronicamente in un sistema informatico.
API
Insieme di regole, protocolli e strumenti che permette a diverse applicazioni software di comunicare tra loro, scambiarsi dati e funzionalit√† in modo standardizzato.
Frontend
La parte di un'applicazione o sito web che l'utente vede e con cui interagisce direttamente (lato client).
Backend
E' la parte "invisibile" di un sito web o applicazione, responsabile della logica, dei database e della gestione dei dati.
